<h2>Guia de Países</h2>
<p>O projeto é parte de uma saga contínua para aprimorar e aplicar meus conhecimentos. Desta vez, foi desenvolvido uma aplicação intuitiva que, a partir da entrada do usuário indicando o nome de um país, realiza uma requisição na API "REST COUNTRIES" (disponível em: https://restcountries.com/). Essa API oferece uma ampla gama de informações sobre países. Na interface da aplicação, essas informações são apresentadas de forma clara e acessível para que o usuário possa consumi-las facilmente. </p>

<h3>Tecnologias Utilizadas</h3>

- HTML
- CSS
- JavaScript
